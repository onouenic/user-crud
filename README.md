# Description

API de CRUD de usuário.

# EndPoints

## /users/

### GET
Retorna todos os usuários

### POST
Cria um usuário

## /users/:id

### GET
Retorna um usário pelo id

### PUT
Atualiza um usuário pelo id

### DELETE
Delete um usuário pelo id


# Installation

```bash
$ npm install
```

# Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
